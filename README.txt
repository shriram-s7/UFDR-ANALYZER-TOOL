# UFDR – Unified Fraud Detection & Response Analysis Tool

## 📌 Project Overview
UFDR Analysis Tool is a unified fraud detection system that analyzes data from multiple sources to identify suspicious patterns and anomalies.  
It helps assess risk levels and detect potential fraud at an early stage.  
The tool supports faster and more informed decision-making to reduce losses and improve security.
